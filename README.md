## 💯프로젝트 목표

- 구글 확장 프로그램인 momentum 클론 코딩
- Babel, Webpack을 이용한 트랜스파일링 및 모듈 번들링 학습
- express 서버를 이용한 간단한 fake server 경험과 CRUD API 구현
- 외부 api 사용과 서버 요청을 하며 axios 경험과 async / await 사용
- git, gitHub, gitflow를 활용한 협업
